# 1. Product Scope & User Personas

## Platform Name: **FlowClass**

FlowClass is a modern, animation-first Learning Management System (LMS) designed to feel more like a fluid SaaS product than a traditional academic tool. It reimagines Google Classroom's workflow while eliminating UX friction points.

---

## User Roles & Core Features

### 👨‍🏫 Teacher
| Feature | Details |
|---|---|
| **Course Management** | Create/manage courses, set cover themes, archive |
| **Stream Posting** | Rich-text announcements, attach files/links, schedule posts |
| **Assignment Creation** | Set due dates, point values, rubrics, topics |
| **Grading Dashboard** | Speed-grader with keyboard nav, bulk feedback templates |
| **Roster Management** | Import students, email class, view engagement metrics |
| **Analytics** | Submission rate trends, grade distribution, late stats |

### 👩‍🎓 Student
| Feature | Details |
|---|---|
| **Dashboard** | Aggregated upcoming work across all courses |
| **Class Stream** | View announcements, assignments, class materials |
| **Assignment Submission** | Upload files, write responses, track draft state |
| **Grade Viewer** | See individual grades, class average, feedback |
| **Calendar** | Visual due-date overview with color-coded courses |
| **Collaboration** | Post comments, ask questions, peer discussions |

### 🔧 Administrator
| Feature | Details |
|---|---|
| **User Management** | CRUD users, bulk import, role assignment |
| **Course Audit** | View all courses, analytics dashboard |
| **System Settings** | SSO config, storage limits, feature toggles |
| **Reporting** | Usage stats, engagement reports, export to CSV |

---

## Essential Views

| View | Roles | Description |
|---|---|---|
| **Dashboard** | All | Aggregated overview, upcoming work, recent activity |
| **Class Stream** | Teacher, Student | Chronological feed of posts with comments |
| **Assignments** | Teacher, Student | List/grid view of all assignments with status |
| **Grading Dashboard** | Teacher | Queue-based grader with submission preview |
| **People / Roster** | Teacher, Admin | Student list with email, last-active indicators |
| **Calendar** | All | Monthly/weekly view with assignment due dates |
| **Settings** | All | Profile, notification prefs, theme toggle |
